# UI Prototype (Next.js)

This is a spec-faithful UI-only prototype designed to be wired to a headless WordPress later.

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Routes
- /works
- /works/[slug]?mode=gallery|index&img=1
- /timeline
- /text
- /text/[slug]
- /text/[slug]/reading
